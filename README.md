# IRC-Server
*This server was designed for learning purposes*

## Description
An IRC server designed in C++ (98), using sockets comunications.
*IRC* stands for Internet Relay Chat. It was develop for the first time in 1980.

My version is a basic server, capable of perfom the basic of comunication, like create channels, invite users, admin functions and more
